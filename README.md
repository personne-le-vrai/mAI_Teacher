# mAI_Teacher - Your 24/7 AI Professor 🎓

## 📝 About the Project
This repository contains the source code for the website of **mAI_Teacher**, a fictitious startup developed as part of the ISEN web project (CIR1 group). 
mAI_Teacher is an AI-powered educational assistant that helps students study. Our platform generates PDF study sheets (exercises and lessons), explains complex concepts both in writing and orally, and allows users to share their sheets for personalized corrections.

## 🚀 Features
The site consists of 6 interactive pages fully driven by JavaScript:
* **Home:** AI presentation with smooth text animations and video integration.
* **About:** The startup's missions and vision presented in interactive modal windows.
* **Products & Services:** A catalog of tutoring offers with a dynamic filtering system via a form, and a Canvas-based animation during purchase simulation.
* **Team:** Presentation of the founders featuring a playful "scratch-off" effect on photos, and a secure Edit mode (admin access) to modify the DOM live.
* **Contact:** A contact form with strict real-time input validation, including a JavaScript mini-game against the computer that must be won to send the message.
* **Find Us (API Bonus):** An interactive access map integrating the WeatherApp API to display local weather, along with GPS coordinate calculations and distance metrics.

## 🛠️ Tech Stack
* **HTML5 & CSS3:** Semantic structure and 100% responsive design.
* **JavaScript (Vanilla):** Event handling (clicks, hovers, forms), DOM manipulation, API requests, and time management (stopwatch, loading delays).
* *Note: This project was built without CSS frameworks like Bootstrap to consolidate native frontend skills*.
